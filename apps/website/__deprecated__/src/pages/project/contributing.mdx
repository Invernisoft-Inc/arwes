# Contributing

> **The project is not fully open for sustancial contributions yet until the
`v1.0.0-beta.1` is released.**

All your contributions are welcome!

There are many ways to help out:

- If you have found a bug, report it on GitHub.
- Work on open issues.
- Add unit tests to increase use case coverage.
- Improve the documentation.
- Blog about how to use the framework.
- Create new tools based on Arwes systems.

When contributing to this repository, please first discuss the change you wish to
make [via GitHub issue](https://github.com/arwes/arwes/issues/new) with the owners
of this repository before making a change.

Have a question or an idea? Share it on [Discord](https://discord.gg/s5sbTkw)!

Please note we have a [code of conduct](/community/code-of-conduct), please follow
it in all your interactions.

Read the following guides to contribute to the project:

- [Contribution workflow](/project/contributing/workflow).
- [Project architecture](/project/contributing/architecture).
- [Contributing to the Play app](/project/contributing/play).
- [Contributing to the Perf app](/project/contributing/perf).
- [Contributing to the website](/project/contributing/website).

Thank you for your contributions!
