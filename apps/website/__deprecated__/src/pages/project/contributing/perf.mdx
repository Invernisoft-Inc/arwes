# Contributing to the Perf app

> The Perf application is still under development.

The [Perf](/perf) is a set of mini web applications to test specific performance
use cases of the framework functionalities.

## Setting up

The repository has to be set up first. See [workflow instructions](/project/contributing/workflow).

Then the website can be installed:

```bash
# at root folder
cd perf
npm install
```

## Running

```bash
# at root folder
cd perf
npm run develop
```
